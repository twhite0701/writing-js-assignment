This drill tests your ability to replace the content of DOM nodes and use variables as arguments to a function.

    Write a function named getDrWho which takes one parameter called who. The function should return a template string that states [someone] is Dr. Who!.

    For example, if the function was to be called like so:

    getDrWho('William Hartnell')

    The return value would be a string of William Hartnell is Dr. Who!

    Use the return value of this function when called with the variable drWho to replace the content of the p element with a class attribute of "who".

Who's Dr. Who?

Peter Capaldi is Dr. Who!
